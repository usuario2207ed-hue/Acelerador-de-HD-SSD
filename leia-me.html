Leia-me (versão simplificada)

Guia rápido para instalar e usar o Painel Acelerador de Disco - EDCELL-TECH no Windows. Siga os passos na ordem.

1. Estrutura mínima da pasta

Coloque estes arquivos juntos na mesma pasta (raiz do projeto):

index.html
server.js
Acelerador de disco.bat
start.bat
Leia-me.html

2. Requisitos

  • Windows 10/11
  • Node.js instalado (baixar em nodejs.org)
  • Permissão de administrador para executar o start.bat

3. Passo a passo — modo automático (recomendado)

  1. Coloque todos os arquivos na mesma pasta.
  2. Clique com o botão direito em start.bat e escolha Executar como administrador.
  3. O start.bat fará automaticamente:
    • criar package.json se faltar;
    • instalar o express (npm install express);
    • iniciar o servidor Node.js (server.js) em uma janela;
    • abrir o navegador em http://localhost:3000;
  4. Na página abra o painel e clique nas opções para executar as ações desejadas.

4. Passo a passo — modo manual

  1. Abra o Prompt de Comando (ou PowerShell) na pasta do projeto.
  2. Execute estes comandos na ordem:
    3. npm init -y
npm install express
node server.js
  3. Abra no navegador: http://localhost:3000

5. O que cada botão faz (resumo)

  • Gerenciador de Dispositivos: abre o devmgmt.msc (ajuste manual de políticas de disco).
  • Instalar DiskMark/PrimoCache: executa o instalador se estiver em setup\Combo Instalador.exe.
  • Reiniciar Computador: reinicia o sistema (shutdown /r /t 5).
  • Teste com CrystalDiskMark: abre o CrystalDiskMark se instalado no caminho padrão.
  • Abrir PrimoCache: abre o PrimoCache para configuração manual.
  • Abrir Leia-me: abre esta página (Leia-me.html).

6. Dicas rápidas e segurança

  • Execute como administrador quando solicitado.
  • Revise o conteúdo do .bat antes de executar (segurança).
  • Se o antivírus bloquear, permita manualmente se confiar no projeto.
  • Se a porta 3000 estiver ocupada, edite server.js para usar outra porta.

7. Problemas comuns

  • Node não encontrado: instale o Node.js e reinicie o terminal.
  • Express não instalado: rode npm install express manualmente.
  • Botões não respondem: verifique se o servidor está rodando (node server.js).
